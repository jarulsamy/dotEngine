## Building

Standard cmake procedure.

1. Clone the repo and `cd` into it
```
$ git clone https://github.com/jarulsamy/dotEngine.git
$ cd dotEngine
```

2. Create a directory for the build
```
$ mkdir build
$ cd build
```

3. Generate the `Makefile` and build
```
$ cmake ..
$ make
```

4. Run the resulting binary
```
$ ./dotEngine
```
